Impotant FAQ

What is the Document Object Model (DOM)?

The Document Object Model is a programming interface that represents the structure of a web page in a way that programming languages like JavaScript can understand and manipulate.

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, use the document.querySelector() method.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where you add an event listener to a parent element instead of each child element. Event delegation is useful because it enhances performance and simplifies code by attaching a single event listener to a parent element instead of multiple listeners to individual child elements. This approach not only reduces memory usage but also ensures that dynamically added elements are automatically covered without requiring additional listeners.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, you can modify its attributes such as id, src, or alt using methods like setAttribute to assign values, getAttribute to retrieve values, and removeAttribute to remove them. For styles, the style property allows you to directly apply CSS rules like changing color, font size, or background. Additionally, the classList property is useful for adding, removing, or toggling CSS classes, making it easier to manage an element's appearance dynamically. These methods enable flexible and efficient updates to the DOM.
